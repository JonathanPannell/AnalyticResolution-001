Automation Tools

 

Record and Play Back Tools

  • Selenium

 

    • Selenium is a portable software-testing framework for web applications. Selenium provides a record/playback tool for authoring tests without the need to learn a test scripting language
    • It also provides a test domain-specific language (Selenese) to write tests in a number of popular programming languages, including C#, Groovy, Java, Perl, PHP, Python, Ruby and Scala
    • Available for Windows, Linux and OS X platforms
  • Selenium IDE
    • Create Quick Bug reproduction scripts
    • Create Scripts to aid in automation exploratory testing
  • Selenium Webdriver -
    • Create robust, browser-based regression automation suites and tests
    • Scale and distribute script across many environments

 

 

API Automation / Metadata Integration Testing

  • SOAP UI

 

Systems transfer meta data between them, common transfer methods include flat files or SOAP. Once the meta data arrives at the receiving system, it is validated or rejected depending on the values within the file.

 

Flat files are easy to see and validate, SOAP files not so much, SOAP UI allows the messages to be trapped and validated.
